# Code of Conduct

## Expectations
To maintain a high-quality, professional open-source environment, all contributors must:
- Be respectful and constructive in all communication.
- Accept constructive criticism gracefully.
- Focus strictly on the technical and design goals of the project.

## Unacceptable Behavior
- Harassment, personal attacks, or discriminatory language.
- Spamming issues or pull requests with low-effort contributions.
- Disrupting the project's workflow with off-topic arguments.

## Enforcement
Violations of this code will result in a warning. Repeated or severe violations will result in a permanent ban from the entire Orbit ecosystem. Report issues privately to the repository owner.
